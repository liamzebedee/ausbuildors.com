# Boards

Monthly snapshot pages for the Aus Buildooors community.

## Intent

The boards exist to give visibility to what community members are building, shipping, and doing each month. They're informal — not a polished showcase, more like a communal fridge door where people pin what they've been up to.

Each board is a single static HTML page for that month (e.g. `2026-apr.html`).

## Structure

Each board has two sections:

### Projects
Cards for things people have shipped or are working on. Each card has:
- A thumbnail (usually the author's X pfp)
- Project title and short blurb
- Visit/watch link
- Author attribution linking to their X profile

### Standup
Casual, line-by-line updates from community members. No structure, no formality — just what's going on in people's lives. Think pub chat, not sprint review.

## Adding content

For now, boards are hand-edited HTML. Just copy an existing card or standup line and fill in the details.

## File naming

`{year}-{month-abbrev}.html` — all lowercase. e.g. `2026-apr.html`, `2026-may.html`.
